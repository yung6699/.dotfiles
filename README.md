# .dotfiles

This is for my mac initial setting

1. default-install.sh
    >  - xcode select 
    >  - zsh shell
    >  - homebrew
    >  - iterm2 setting
    >  - mac application


2. node-install.sh
    > - nvm
    > - npm i -global


3. dev-install.sh (docker)

# Install
```
# dotfiles clone
git clone https://github.com/yung6699/.dotfiles.git

# move 
cd .dotfiles

# run .sh files
./default-install.sh
./node-install.sh
```
